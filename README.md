# موريسكو كافيه - نظام إدارة متقدم

## 🚀 نظرة عامة

نظام إدارة شامل ومتطور لمقهى موريسكو، مصمم لتوفير حلول احترافية لإدارة الموظفين، الحضور، الرواتب، المصروفات، والتقارير. النظام مبني بأحدث التقنيات ويوفر واجهة مستخدم حديثة ومتجاوبة.

## ✨ المميزات الجديدة المضافة

### 🎨 تصميم حديث ومتطور
- **واجهة مستخدم محسنة**: تصميم عصري مع ألوان متدرجة وتأثيرات بصرية جذابة
- **تجربة مستخدم محسنة**: تنقل سلس وواجهات بديهية
- **تصميم متجاوب**: يعمل بشكل مثالي على جميع الأجهزة (الهواتف، الأجهزة اللوحية، الحواسيب)
- **أيقونات وألوان متناسقة**: نظام تصميم موحد مع ألوان موريسكو كافيه

### 👥 نظام إدارة الموظفين المحسن
- **ملفات موظفين شاملة**: عرض تفصيلي لجميع بيانات الموظف
- **نظام ورديات متقدم**: دعم للدوام الكامل والورديات المنفصلة
- **حساب ذكي للأجور**: حساب تلقائي للأجور حسب نوع التوظيف
- **إدارة الملاحظات**: نظام متقدم لتسجيل وإدارة ملاحظات الموظفين
- **نظام الإضافي والخصم**: إدارة شاملة للمكافآت والخصومات

### 📊 نظام تقارير احترافي
- **تقارير A4 احترافية**: تصميم مطبوعي عالي الجودة
- **تقارير شاملة**: تغطية كاملة لجميع جوانب العمل
- **تصدير PDF**: إمكانية الطباعة والتصدير
- **إحصائيات تفاعلية**: رسوم بيانية وجداول منظمة
- **متوافق مع الأجهزة المحمولة**: عرض مثالي على جميع الشاشات

### ⏰ نظام ورديات ذكي
- **دعم أنواع التوظيف المختلفة**:
  - **دوام كامل**: يومية واحدة من الصباح للمساء
  - **نظام ورديات**: حساب يوميتين منفصلتين للورديتين
  - **دوام جزئي**: حسب ساعات العمل
- **حساب تلقائي للأجور**: حسب نوع الوردية وساعات العمل
- **إدارة الإضافي**: حساب ساعات العمل الإضافي
- **تتبع الحضور**: سجلات مفصلة للحضور والانصراف

### 💰 نظام مالي متكامل
- **إدارة الرواتب**: حساب وتوزيع الرواتب
- **نظام السحوبات**: تتبع سحوبات الموظفين
- **الإضافي والخصم**: إدارة المكافآت والخصومات
- **تقارير مالية**: تحليل شامل للوضع المالي

## 🛠️ التقنيات المستخدمة

### Frontend
- **React 18**: مكتبة واجهة المستخدم
- **TypeScript**: برمجة آمنة ومتطورة
- **Tailwind CSS**: تصميم سريع ومتجاوب
- **shadcn/ui**: مكونات واجهة مستخدم عالية الجودة
- **Lucide React**: أيقونات جميلة ومتسقة

### Backend & Database
- **Supabase**: قاعدة بيانات PostgreSQL في السحابة
- **Row Level Security (RLS)**: أمان متقدم للبيانات
- **Real-time subscriptions**: تحديثات فورية
- **Authentication**: نظام تسجيل دخول آمن

### Additional Tools
- **React Router**: إدارة التنقل
- **React Query**: إدارة حالة البيانات
- **React Hook Form**: إدارة النماذج
- **Date-fns**: معالجة التواريخ
- **React Hot Toast**: إشعارات المستخدم

## 🚀 كيفية الاستخدام

### التثبيت والإعداد
```bash
# استنساخ المشروع
git clone [repository-url]
cd cafe-hub-main

# تثبيت التبعيات
npm install

# إعداد متغيرات البيئة
cp .env.example .env.local
# تعديل ملف .env.local بمعلومات Supabase

# تشغيل المشروع
npm run dev
```

### إعداد قاعدة البيانات
1. إنشاء مشروع Supabase جديد
2. تشغيل ملفات الهجرة في مجلد `supabase/migrations`
3. إعداد RLS policies
4. إضافة بيانات الموقع الافتراضية

### الاستخدام الأساسي
1. **إدارة الموظفين**: إضافة وتعديل بيانات الموظفين
2. **نظام الورديات**: جدولة وإدارة الورديات
3. **تسجيل الحضور**: تتبع حضور وانصراف الموظفين
4. **إدارة الرواتب**: حساب وتوزيع الرواتب
5. **التقارير**: إنشاء تقارير شاملة
